**Synaptive** is a modern, engineering-first digital agency portfolio. We strip away the fluff to focus on what matters: clean, scalable code and high-performance web experiences.

:link: **Live Demo:** [synaptive.space](https://synaptive.space/)

## :art: Design Philosophy

The site utilizes a "Cinematic Yellow" neo-brutalist aesthetic (`#EAB308` & `#121212`) combined with high-performance glassmorphism. It is built to demonstrate technical capability through:

* **Interactive 3D Elements:** A custom Three.js robot hero section that tracks mouse movement.
* **Smooth Motion:** Integrated `Lenis` for inertia-based smooth scrolling.
* **Bento Grid Layout:** A responsive, browser-framed service showcase.
* **Deployment Carousel:** A custom swipe-enabled 3D card carousel for recent work.

## :tools: Tech Stack

* **Core:** Semantic HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Styling:** Tailwind CSS (via CDN) + Custom CSS for glass effects and animations.
* **3D Graphics:** Three.js.
* **Icons:** Lucide Icons.
* **Backend:** Google Apps Script (Serverless Form Handling).

## :open_file_folder: Project Structure

```bash
synaptive/
├── index.html      # Main markup
├── style.css       # Custom animations, glass effects, and overrides
├── script.js       # Logic for Three.js, Carousel, and Form Validation
└── assets/         # Project images (Create this folder and add your images)

```
